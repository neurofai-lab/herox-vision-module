# ARISE context

This repository packages the HERoX vision capability as a reusable HRI module. It extracts the human/body perception part of the demonstrator into an open ROS 2 package that can be inspected, built, launched, and adapted independently of the full HERoX system.

The module contributes to ARISE by providing a reusable perception component for human-aware robotic systems. It publishes human body detections and positions in ROS 2 and ROS4HRI-style topics, enabling other components to consume human state information for operator awareness, safety-aware behaviour, human-aware navigation, or task assistance.
